# Color Prediction Game Platform

Ek modern, responsive aur feature-rich color prediction platform.

## Pariyojana ka Parichay (Project Overview)
Yeh ek cross-platform gaming app hai jahan users time-based periods par apne colors aur numbers predict kar sakte hain. Wingo style prediction game logic ke saath, yeh app bilkul seamless mobile experience deta hai.

## Features List
- **Wallet Section:** Apna balance check karein, deposit aur withdraw options.
- **Time-Based Periods:** Live reverse countdown timer jisme last 10 seconds mein betting block ho jati hai.
- **Betting Area:** Green, Violet, aur Red colors; 0 se 9 tak numbers; aur Big / Small options.
- **Game History & Charts:** Pichle periods ke results dekhein aur data trends ka analysis karein.

## Tech Stack
- Frontend: Flutter (Cross-platform support ke liye).
- Responsive UI: Mobile browsers, Android aur iOS ke liye optimized.

## Kaise Run Karein (How to run)
```bash
flutter pub get
flutter run
```
